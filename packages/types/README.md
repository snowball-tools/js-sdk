# @snowballtools/types

This package is responsible for providing types and interfaces for the Snowball SDK.

## Installation

### NPM

```zsh
npm install @snowballtools/types
```

### Yarn

```zsh
yarn add @snowballtools/types
```

## Usage

```typescript
import type { Chain } from '@snowballtools/types';
```
