# @snowballtools/utils

## Description

This package is responsible for handling snowballtools sdk utils

## Installation

```zsh
# NPM
npm install @snowballtools/utils

# Yarn
yarn add @snowballtools/utils

# PNPM
pnpm add @snowballtools/utils

# Bun
bun install @snowballtools/utils
```
