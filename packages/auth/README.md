# @snowballtools/auth

## Description

This package is responsible for handling user authentication (who you are) and authorization (what you can do).

## Installation

### NPM

```zsh
npm install @snowballtools/auth
```

### Yarn

```zsh
yarn add @snowballtools/auth
```

## Usage

```typescript
import { Auth } from '@snowballtools/auth';
```
