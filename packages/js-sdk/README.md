# @snowballtools/js-sdk

## Installation

```zsh
# NPM
npm install @snowballtools/js-sdk

# Yarn
yarn add @snowballtools/js-sdk

# PNPM
pnpm add @snowballtools/js-sdk

# Bun
bun install @snowballtools/js-sdk
```
